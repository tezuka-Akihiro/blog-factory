# Phase 4: X告知文生成

記事完成後、Xへの告知ポストを生成する。

## テンプレート

```
【Xのトレンドをしっかり深堀り】

[トレンド名]について、ClaudeにClaudeMixのターゲット向けに聞いてみました。

✅ [読者が得られる気づき1]
✅ [読者が得られる気づき2]
✅ 今週中に取れるアクション

👇 全文はこちら
[記事URL]

#ClaudeMix #[関連ハッシュタグ1] #[関連ハッシュタグ2]
```

## テンプレートの埋め方

- **[トレンド名]**: 記事タイトルに使ったキーワード（短く）
- **[気づき1・2]**: セクション2・3から、読者が「おっ」となる発見を2つ選ぶ。14文字以内で端的に
- **[記事URL]**: `https://claudemix.com/blog/[slug]`
- **[ハッシュタグ]**: 2〜3個。`#ClaudeMix` は固定。残りはトレンド関連タグを選ぶ

## 完成例

**例1（AIモデル発表）:**
```
【Xのトレンドをしっかり深堀り】

Claude 4発表について、バイブコーダー視点でClaudeに聞いてみました。

✅ コンテキスト拡張で「設計→実装」の分断が消える
✅ エージェント強化で「指示出し力」がより重要になる
✅ 今週中に試すべき新機能ベスト3

👇 全文はこちら
https://claudemix.com/blog/x-trend-claude-4-launch

#ClaudeMix #バイブコーディング #Claude4
```

**例2（規制・法律）:**
```
【Xのトレンドをしっかり深堀り】

EU AI Act施行細則確定について、個人開発者への影響をClaudeに聞きました。

✅ オープンソースモデルの「例外規定」の実態
✅ 日本から欧州向けにSaaSを売る場合の注意点
✅ 今すぐ確認すべきコンプライアンスチェック3項目

👇 全文はこちら
https://claudemix.com/blog/x-trend-eu-ai-act-enforcement

#ClaudeMix #EUAI法 #個人開発
```

**例3（NG例）:**

```
新しい記事を書きました。よかったら読んでみてください。
https://...
```
→ 問題: 読者が得られる価値が伝わらない・クリックする理由がない

## 完了条件

- テンプレートの全プレースホルダーが埋まっている
- 気づきが2つとも14文字以内で端的に書かれている
- ハッシュタグが2〜3個選ばれている

生成したポスト文をオペレーターにそのままコピペできる形式で出力する。

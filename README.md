# blog-factory

「技術本 2.0」を製造・メンテナンスし続ける、構造的品質の司令塔。

## claude mix - リポジトリ構成

1 ClaudeMix

有料ブログ販売システム
マークダウン記事のプレビルドも行う。
当初はここで全てを実施していて、リポジトリを分離している。
責務が多すぎるので減らしたい。

2 blog-factory

ブログの編集室
ブログの執筆や、記事全体の編集、サマリー集計などを行う。
経営アドバイザーへの資料作成を新たに追加。

3 img-fortress-processor

サムネイルの印刷と展示
画像のリサイズ・変換・R2へのアップロードを行う。

# Phase 2: AI分析

Phase 1 で検証済みのトレンド情報を元に、4セクション分の分析を構造化する。
以下のプロンプトテンプレートを使って、各セクションに必要な素材を準備する。

## 分析プロンプトテンプレート

以下の問いに対して、ClaudeMix読者（バイブコーダー・技術者・個人事業主）の視点から回答を生成する。

```
## トレンドの概要
[Phase 1 で確認した内容を記載]

## 分析設問

### セクション1用（状況把握）
- 何がいつ起きたか（事実の時系列）
- 世間の主要な反応・論点は何か

### セクション2用（影響分析）
- バイブコーダー・個人事業主への具体的な脅威は何か
- 逆に生まれる機会・メリットは何か
- 「自分ごと化」できる最もわかりやすい事例は何か

### セクション3用（本質探求）
- 表面的な議論の裏にある本質的な構造変化は何か
- 情報の信憑性・誇張されている部分はないか
- 今後3〜6ヶ月で起こりうる最悪/最善のシナリオは何か

### セクション4用（アクションプラン）
- 読者が「今週中に取れる最も有効なアクション」は何か（具体的に1〜3つ）
- このトレンドを踏まえた実用的なプロンプト例を1つ作成してほしい
```

## 信頼性の区別（出力時に必ず適用）

分析結果を整理する際は、以下を明確に区別してメモする。

| 種別 | ラベル |
| :--- | :--- |
| 公式発表・報道から得た事実 | `[事実]` |
| AI分析・予測 | `[AI分析]` |
| 筆者の推論 | `[推論]` |

## 完了条件

4セクション分の分析メモが揃ったら、以下の形式で出力してオペレーターに提示する。

```
【分析メモ】

[セクション1] 状況把握
- [事実] ...
- [事実] ...

[セクション2] 影響分析
- [事実/AI分析] ...（脅威）
- [AI分析] ...（機会）

[セクション3] 本質探求
- [AI分析] ...（本質）
- [推論] ...（リスク予測）

[セクション4] アクションプラン
- アクション1: ...
- アクション2: ...（任意）
- プロンプト例: ...

→ Phase 3（記事執筆）に進みます。
```

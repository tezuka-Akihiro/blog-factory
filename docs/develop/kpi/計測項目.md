# ClaudeMix マーケティングKPI候補一覧

## 1. 認知・流入（トラフィック）

- インプレッション数（検索・SNS）
- ページビュー数（PV）
- ユニークユーザー数（UU）
- クリック率（CTR）
- 指名検索数（「ClaudeMix」等のブランド検索）

## 2. 信頼・エンゲージメント（教育）

- 平均滞在時間
- 読了率
- 直帰率 / 回遊率
- 再訪率（リピート率）
- RSS/SNSフォロー数

## 3. リード獲得・比較検討（防衛拠点）

- 無料ニュースレター登録数
- Lighthouse スコア（100点維持の確認）
- 資料ダウンロード・サンプルコード参照数
- 問い合わせ・相談件数

## 4. コンバージョン・収益（契約）

- 有料会員新規登録数
- コンバージョン率（CVR）
- 顧客獲得単価（CPA）
- 継続率（リテンション率）
- 解約率（チャーンレート） ※最重要
- LTV（顧客生涯価値）
- 月間経常収益（MRR）

## 5. 独自指標（AI共進化・戦略的証明）

- AI学習用構造化データのインデックス数
- ドッグフーディングによる実績公開数（エッジ動的OGP等）
